# IMDb Movie Review Sentiment Analysis using LSTM

An end-to-end Deep Learning project that classifies movie reviews into positive or negative sentiments using an LSTM (Long Short-Term Memory) network trained on the IMDb dataset. The project features an interactive web interface built with Streamlit.

## 🚀 Features
* **Deep Learning Model:** Built using TensorFlow/Keras with LSTM layers to capture text sequence dependencies.
* **Preprocessing Pipeline:** Custom text cleaning (HTML tag removal, regex filtering) and tokenization.
* **Web Interface:** User-friendly web app deployed via Streamlit for real-time inference.

## 📁 Project Structure
* `app.py`: The Streamlit web application interface.
* `data_preprocessing.py`: Script handles data loading, cleansing, and text preprocessing tokenization.
* `model.py`: Handles model building, evaluation, and performance visualization.
* `requirements.txt`: Configuration file containing required external libraries.

## 🛠️ Tech Stack
* **Frameworks:** TensorFlow, Keras, Streamlit
* **Libraries:** NumPy, Pandas, Scikit-learn, Matplotlib, Seaborn
* **Environment:** Kaggle (Training), VS Code (Development)
